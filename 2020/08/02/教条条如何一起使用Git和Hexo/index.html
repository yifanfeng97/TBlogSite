安装Node.js

安装Node.js的目的是为了，给网站提供解释前端的web框架。

首先我们打开Node.js的官网，下载左边的12.18.3的版本。

然后双击下载好的安装包，进行安装。每个界面都使用默认的选项，不做任何更改，直接点击下一步，弹出请求权限的时候点同意，直到安装完成。

恭喜Node.js安装成功！🎈🎈🎈
